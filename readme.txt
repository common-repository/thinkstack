== Description ==

WordPress plugin to embed Thinkstack chatbot in your WordPress site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/thinkstack` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Thinkstack settings page to configure the plugin.
